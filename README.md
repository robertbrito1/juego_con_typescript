# 🎮 Juego en TypeScript

¡Bienvenido al repositorio de mi juego desarrollado en clase, ahora mejorado y reescrito en TypeScript! 🚀

## Descripción

Este proyecto es una adaptación de un juego originalmente creado en JavaScript durante una clase pensado por mi profesor hugo  , con el objetivo de aprender y practicar los conceptos fundamentales de programación. Esta versión utiliza TypeScript para aprovechar las ventajas de un sistema de tipos estático, mejor legibilidad y mayor mantenibilidad del código.

## Características

- 🔄 Migración total de JavaScript a TypeScript.
- 💡 Código más seguro y fácil de depurar gracias a los tipos.
- 🎨 Interfaz y lógica de juego fiel al proyecto original.
- 📝 100% realizado como parte del proceso de aprendizaje en clase.

## Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/robertbrito1/juego_con_typescript.git
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Para compilar el juego:
   ```bash
   npm run build
   ```
4. ¡Ejecuta y disfruta del juego en tu navegador!

## ¿Por qué TypeScript?

Pasar el proyecto a TypeScript permite:

- Detectar errores antes de ejecutar el juego.
- Mejorar la estructura y organización del código.
- Facilitar futuros mantenimientos y nuevas características.

## Estructura del Proyecto

- `/src` — Código fuente en TypeScript.
- `/dist` — Versión compilada lista para ejecutar.
- `/assets` — Recursos gráficos y de sonido (si aplica).
